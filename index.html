Project Overview

Our team developed an Online Restaurant Ordering System (OROS) API that allows customers to place orders without creating an account. The system supports menu browsing, order tracking, and promotional codes.

Key Features

  • Create, update, and delete menu items
  • Place orders as a guest
  • Track order status
  • Apply promotional codes
  • View revenue and order data
